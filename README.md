# 8BitDoRMKFirmware

## This project has been placed on pause due to me having to return the keyboard due to hardware failure, meaning I will have no way of testing out the resulting product.

This decompilation was done on firmware for 8BitDo Retro Mechanical Keyboard 85HE, v1.3.6.

This keyboard uses the Telink chip: **TLSR8273F512ET48**
